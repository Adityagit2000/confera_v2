## 🚀 Getting Started

To run this project locally, you'll need **Node.js** and **npm** installed. We recommend using [nvm](https://github.com/nvm-sh/nvm#installing-and-updating) for managing Node versions.

Follow these steps:

```sh
# Step 1: Clone the repository
git clone <YOUR_REPO_URL>

# Step 2: Navigate into the project directory
cd <YOUR_PROJECT_NAME>

# Step 3: Install dependencies
npm install

# Step 4: Run the development server
npm run dev
